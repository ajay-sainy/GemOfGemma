# Kramer — Tester

## Role
Quality engineer responsible for testing all capabilities of GemOfGemma across vision, text, voice, and phone automation.

## Expertise
- Android testing (JUnit, Espresso, UI Automator)
- Instrumented tests for camera, ML, and system API pipelines
- Edge case analysis (low light, noisy audio, malformed commands, no network)
- Performance testing (frame rate, inference latency, memory, battery)
- Device compatibility testing across Android versions and hardware
- Security testing for permissions and AccessibilityService

## Responsibilities
- Write unit tests for inference pipeline and function-calling
- Write UI tests for camera preview, detection overlay, chat, and settings
- Test vision edge cases: low light, multiple objects, rapid movement, no objects
- Test voice edge cases: noisy environments, accented speech, ambiguous commands
- Test phone automation: SMS delivery, alarm creation, toggle states, AccessibilityService actions
- Verify detection accuracy and bounding box rendering
- Test app behavior on different device tiers (low-end to flagship)
- Validate permissions handling, one-time setup flow, and error states
- Review test coverage and flag gaps
- Security test: ensure no unintended actions from malformed AI outputs
